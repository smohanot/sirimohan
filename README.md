# HireHub

HireHub is a recruitment management platform designed to streamline the hiring process for companies and job seekers. The application enables organizations to manage job postings, applications, and candidate information efficiently while providing job seekers with an easy way to explore opportunities and apply for positions.

Built using **Spring Boot**, HireHub provides secure authentication, RESTful APIs, and scalable backend architecture for managing recruitment workflows.

## Features

* User authentication and authorization using JWT
* Job posting and management
* Candidate registration and profile management
* Application tracking for recruiters
* Secure REST APIs for frontend integration
* Role-based access control for recruiters and applicants

## Tech Stack

* **Backend:** Java, Spring Boot
* **Security:** Spring Security, JWT
* **Database:** MySQL / PostgreSQL
* **Build Tool:** Maven
* **Version Control:** Git & GitHub
